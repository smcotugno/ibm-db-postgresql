Terraform and Schematic example of IBM Cloud Database PostgresSQL

## Database Terraform example

This example shows how to Create a PostgreSQL database


To run, configure your IBM Cloud provider

Running the example

For planning phase

```shell
terraform plan
```

For apply phase

```shell
terraform apply
```

For destroy

```shell
terraform destroy
```
